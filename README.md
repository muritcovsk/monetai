# MonetAI 💰

Assistente de investimentos com inteligência artificial — explica o mercado financeiro em português simples para iniciantes.

## 🌐 Acesse
[monet-ai.netlify.app](https://monet-ai.netlify.app)

## 🤖 O que é
O MonetAI usa IA para educar brasileiros sobre investimentos sem jargão técnico. O usuário conversa com a MAI — uma assistente de IA — e aprende sobre ações, FIIs, renda fixa, Selic e muito mais.

## ✨ Funcionalidades
- Onboarding personalizado com 3 perguntas
- Dashboard com dados reais de mercado (Ibovespa, Dólar, Selic, Bitcoin)
- Ações em destaque com preços em tempo real
- Chat com IA especializada em investimentos
- Cards educativos sobre conceitos financeiros

## 🛠️ Tecnologias
- HTML, CSS e JavaScript puro
- API Groq (LLaMA 3.1)
- API Brapi (dados de mercado brasileiros)
- API AwesomeAPI (câmbio)
- API Banco Central do Brasil (Selic)
